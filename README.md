# Paraíso · Members Portal

Private fractional villa ownership · Dominican Republic.

## Deploy in 2 minutes (no terminal, no installs)

1. Go to **[vercel.com/signup](https://vercel.com/signup)** and create a free account (use Google or GitHub login — fastest).
2. After signup, click **"Add New..."** → **"Project"** at the top right.
3. You'll see a button "**Import**" or "**Browse All Templates**". Look for a small link that says **"Import Third-Party Git Repository"** OR drag this entire folder (the unzipped one) into Vercel's "Deploy" area.
   - If Vercel asks for a Git repo, the simplest path is: go to [github.com/new](https://github.com/new), create a new public repo, drag this folder's contents into it (use the GitHub "upload files" link), then come back to Vercel and import it.
4. Vercel auto-detects this is a Vite project. Just click **"Deploy"**.
5. ~60 seconds later you get a live URL like `paraiso-portal-abc123.vercel.app`. Click "Visit" and send the link to your partner.

## Login

Demo mode: type any email and password, or click any of the 8 member names at the bottom of the login screen to enter instantly.

## What's inside

- React + Vite (zero config)
- One file (`src/App.jsx`) contains the entire portal
- No backend, no database, no API keys — pure prototype
